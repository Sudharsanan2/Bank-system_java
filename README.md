This Java code is a simple simulation of a banking system. It demonstrates object-oriented programming (OOP) concepts such as encapsulation, classes, objects, and data abstraction. 

# Purpose of the Code:

To simulate a basic banking system that can,

Create multiple customer accounts.

Perform deposits into those accounts.

Display account information.

Handle invalid operations like depositing into a non-existent account.

# Why use this code?

This code is helpful for,

Learning OOP concepts in Java.

Understanding data storage using HashMap.

Practicing basic user-defined class creation (Account, Bank).

Simulating real-world systems like banking in code.
